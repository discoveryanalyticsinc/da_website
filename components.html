Alert and info boxes

Alert boxes for various message states: Information, Success, Warning and Error.

Info Just to inform you. To create this info box any text, or tags in a div with a class of "allert-info"

Success Account created! To create this success box wrap any text, or tags in a div with a class of "allert-success"

Warning! Insecure password. To create this warning wrap any text, or tags in a div with a class of "allert-warning"

Error! Terrible, terrible damage! To create this error box wrap any text, or tags in a div with a class of "allert-error"

Buttons showcase

To insert a button add "button" class to "a" tag. You can control the size of button by adding more classes - "large", "medium" or "small". To choose simple button add "simple" class.

Large button Medium button Small button

To use the shiny buttons like those below just change the "simple" class to "shiny".

Large button Medium button Small button

To change button color go to the main.css file, find the ".simple" or ".shiny" declaration and change the "background-color" or "background" property with a desired color value.
